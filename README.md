# 🐾 CatTinder — приложение для свайпа котиков

**CatTinder** — Flutter-приложение с Tinder-подобным свайп-интерфейсом для просмотра котиков.

Приложение подбирает случайного кота, позволяет свайпать карточки, отслеживает лайки и генерирует имя с помощью AI-модели по фото.

---

## Реализованные фичи

### Карточка кота
- Загрузка случайных котиков из **TheCatAPI**
- Анимация свайпа вправо/влево
- Лайки отображаются в реальном времени
- Тап по карточке → детальный экран

### AI-генерация имени
- Интеграция с **OpenRouter (GPT)**
- Генерация уникального имени по фото
- Имя отображается на главной и детальной карточке

### Экран пород (Breeds)

- Загрузка списка пород из TheCatAPI
- Умный fuzzy-поиск по названию и стране происхождения
- Красивые карточки пород с градиентами и тенью
- Переход на подробный экран породы по нажатию
- Детальная информация: описание, страна, темперамент и ⭐ рейтинг характеристик

### Информация о породе
Если у котика есть порода — показываются:
- описание  
- страна происхождения  
- темперамент  
- интеллект / ласковость / энергичность / социализация  

Если породы нет — показывается красивый fallback-текст.

### Интерфейс
- Градиентный фон
- Плавные анимации
- Скруглённые карточки и тени
- Анимация лайка (пульсация)
- Аккуратный детальный экран кота
